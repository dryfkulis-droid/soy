# Beauty by Soyka

Gotowa statyczna strona dla salonu Beauty by Soyka w Siedlcach.

## Pliki

- \`index.html\` - główna strona z hero, ofertą, cennikiem, opiniami, FAQ i kontaktem.
- \`styles.css\` - style responsywne, animacje, dark mode i wygląd premium.
- \`script.js\` - cennik, FAQ, animacje przewijania, menu mobilne i cookie banner.
- \`privacy.html\` - prosta polityka prywatności / RODO.
- \`404.html\` - strona błędu dla GitHub Pages.

## Publikacja na GitHub Pages

1. Wrzuć pliki do repozytorium GitHub.
2. Wejdź w \`Settings\` -> \`Pages\`.
3. Wybierz \`Deploy from a branch\`.
4. Ustaw branch \`main\` i folder \`/root\`.
5. Zapisz ustawienia.

## Do podmiany przed finalnym wdrożeniem

- Zdjęcia w sekcji hero i galerii.
- Adres e-mail w formularzu kontaktowym.
- Ewentualny link do Instagrama, Messengera i WhatsApp.
- Treść polityki prywatności po konsultacji z osobą odpowiedzialną za RODO.
